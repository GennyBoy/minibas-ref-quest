# 🏀 ミニバスTO・審判クエスト

ミニバスケットボール（U12）の **TO（テーブルオフィシャルズ）** と審判のルールを、クイズゲーム形式で学べるWebアプリです。

**▶ アプリを開く: https://gennyboy.github.io/minibas-ref-quest/**

## 特徴

- **役割別に学べる**: スコアラー / Aスコアラー / タイマー / SCオペレーター / TOサポーター の5役割から選んで、その役割に必要な知識だけをクイズで練習
- **間隔反復**: 間違えた問題は翌日に、覚えた問題は少しずつ間隔を空けて再出題（SM-2簡易版）
- **バッジシステム**: 役割ごとの習熟度70%でバッジ獲得、全役割制覇でTOマスター
- **U12/一般の切り替え**: ミニバス固有ルール（6分Q・45秒タイムアウト・フロントコートなし等）と一般ルールを区別して出題
- **PWA対応**: スマホのホーム画面に追加すればオフライン（体育館）でも使える
- 進捗は端末内（localStorage）にのみ保存。バックアップのエクスポート/インポート機能つき

## 開発

```bash
npm install
npm run dev              # 開発サーバー
npm test                 # ユニットテスト（SRS/XP/習熟度）
npm run validate-content # 問題データの検証
npm run build            # 検証+型チェック+ビルド
```

問題データは `content/questions/*.ts` にあります。`main` へのpushでGitHub Actionsが自動デプロイします。
