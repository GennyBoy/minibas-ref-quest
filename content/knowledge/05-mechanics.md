## 審判の役割分担（2人制）

ミニバスは通常2人制審判。

| 役割 | 呼び名 | 位置 |
|-----|------|------|
| リード審判 | Lead | エンドライン付近（ゴール下） |
| トレイル審判 | Trail | ハーフライン付近（攻撃方向後ろ） |

---

## 基本的な立ち位置

### 攻撃時（一般的な半面オフェンス）

```
        [バスケット]
             |
      ┌──────┴──────┐
      │   ペイント   │
      └─────────────┘
           ↑
        Lead審判
   （エンドラインの外、斜め45°）

      
      
                          Trail審判
                   （フリースローライン延長付近）
```

### ポイント
- Leadは**コーナーからコーナーへの動線**を意識して動く
- Trailは**ペネトレイト・ジャンプシュート**を主に見る
- 2人が**同じアングルに重ならない**ようにする（オフィシャルズ・マニュアル参照）

## 主要シグナル一覧

| シグナル | ジェスチャー |
|---------|-----------|
| タイムアウト | 両手で「T」の字 |
| ファウル（コール） | 片腕を上げて拳を握る |
| チャージング | 手を胸の前で前方に押し出す |
| ブロッキング | 両手を腰に |
| バイオレーション | 下方向に指を振る |
| トラベリング | 両腕を前後に回転 |
| ダブルドリブル | 両手を交互に下へ |
| アウトオブバウンズ | 方向を示す |
| フリースロー（本数） | 指の数で示す |

> ⚠️ 正式なシグナルはオフィシャルズ・マニュアルで確認。上記は概略のみ。

## コミュニケーション

### 審判同士のコミュニケーション
- アイコンタクトで状況を確認
- 「ダブル（Dual）」コール（2人が同時にコール）は避ける
- オフサイドのプレーはもう一方に確認

### テーブルオフィシャルズ（TO）との連携
- ファウルの報告：番号・ファウルの種類・フリースロー本数（TO側はアイコンタクト→同時復唱→サムアップで受ける）
- タイムアウトの確認
- スコアシートへの記録確認

> TO側の業務そのものは [[07-to-basics]]（概要）・[[08-to-scoresheet]]（スコアラー）・[[09-to-timer-shotclock]]（タイマー・ショットクロック）を参照。

## 試合進行

### 試合開始
1. TOとのブリーフィング（スコアシート確認）
2. コイントス（先攻を決める）
3. ジャンプボールで試合開始

### ピリオド間
- ベンチへの指示・確認
- チームファウル数の確認

### タイムアウト
- コーチがリクエスト → 次のデッドボール時に認める
- TOにサインを出す
