# タイマー

## 任務の全体像

- 競技時間・タイムアウト・プレーのインターバルをはかる
- 各Q・OTの終了をブザーで知らせる（ブザーが鳴らない／聞こえないときは**笛**で代替）
- 常にゲームクロックの**スタート／ストップボタンに手を置いておく**
- タイムアウト計測は**ゲームクロックとは別のストップウォッチ**で行う

## ゲームクロックのスタート／ストップ

### スタートする瞬間

| 場面 | スタートのタイミング |
|------|---------------------|
| ジャンプボール | ジャンパーが**正当にボールをタップしたとき** |
| スローイン | コート上のプレーヤーが**ボールに触れた（触れられた）とき** |
| 最後のFTが不成功でボールがライブ | リングで弾んだボールに**コート上のプレーヤーが触れたとき** |

- 触れた瞬間が確認できないときは、審判の**手を振り下ろすシグナル**でスタート
- 国内の作法：スタート前に手のひらを開いて上げておき、**スタートと同時に手を握って下ろす**。「スタート」と声に出し、ショットクロックオペレーターと同時にコール、目視で動作確認
- 各Q・OT開始時は、コート上に**5人ずついるか確認してから**手を上げる

### ストップする瞬間

| 場面 | 補足 |
|------|------|
| ボールがライブで**審判が笛を鳴らしたとき** | 笛が聞こえにくいときは審判のシグナルを注視 |
| Q・OTの競技時間終了（自動で止まらなかったとき） | |
| **タイムアウトを請求しているチームの相手がフィールドゴールで得点したとき** | 速やかに止めてブザー |
| ショットクロックのブザー後、**審判が笛を鳴らしたとき** | 審判が笛を鳴らさなければ止めない |
| 第4Q・OT残り2:00以下でフィールドゴール成功（一般のみ） | **U12にはこの規定なし** → [[04-minibasket-diff]] |

- ストップと同時に「ストップ」とコールし、**開いた手を頭上に上げる**。すぐ再開しないときは一旦下ろし、再開準備（5人揃った等）を確認して再び上げる。フリースローは**最後の1投の前**に手を上げる
- 止め忘れ・動かし間違いをしたら、**審判に知らせてから**クロックを修正する

## タイムアウトの計測

- **審判が笛を鳴らしてタイムアウトのシグナルを示したとき**からストップウォッチで計測（スコアラーのブザーが起点ではない）

| | 一般 | U12 |
|---|------|-----|
| 長さ | 60秒 | **45秒** |
| ブザー | **50秒**経過時と**60秒**経過時 | **35秒**経過時と**45秒**経過時 |

- ブザーのスイッチがタイマーの位置にないときはスコアラーが鳴らす

## インターバルの計測とブザー

| 対象 | 一般 | U12 |
|------|------|-----|
| Q1-Q2間・Q3-Q4間 | 2分（終了30秒前にブザー） | **1分（途中のブザーなし・終了時のみ）** |
| ハーフタイム | 10分（国内推奨）。終了3分前・1分30秒前にブザー | **5分。終了3分前・1分前にブザー** |
| OT間 | 2分 | 2分（**終了1分前にブザー**） |
| ゲーム開始前 | 20分（開始3分前・1分30秒前にブザー） | 同様（主催者判断で短縮あり） |

- インターバルが終了したらブザーと同時にゲームクロックを次のQの時間（U12は6:00、OTは3:00）にリセット

## ゲーム中の声出し（タイマー系コミュニケーション）

- 残り時間の定期コール：各Q**残り3分と残り1分**をクルーに知らせる
- 再開前：手を上げて残り時間をコール（例：3分12秒 →「さんいちに」）。手を上げたあとはベンチを見ない
- **「14秒」コール**：ショットクロックが14秒になったら「14秒」と伝える（誤リセット時や14秒リセット判断のとき、既に14秒未満かを2人で確認するため）
- ショットクロック残り5秒からのカウントダウン（「5、4、3…」）はタイマーが担当
- Q終わりにゲームクロックとショットクロックのカウントダウンが重なるときは、**タイマー＝ゲームクロック、SCオペレーター＝ショットクロック**を分担してカウント
- 一般では第4Q・OTの**残り2分10秒から「10、9…0」とカウント**し、L2M（最後の2分）に入ったことを全員で共有する（U12はL2M規定なし）

## 機器故障時

- ゲームクロックが動かないときは、タイムアウト用ストップウォッチで計測を続け、立ち上がるなどして審判に気づかせ、ボールがデッドになったときに「何秒間止まっていたか」を伝える
- クロックのブザーが鳴らない／聞こえないときは、Q終了と同時に**笛**を鳴らす

# ショットクロックオペレーター

## 大前提：ボールの「コントロール」を見極める

ショットクロックは**チームコントロール**とともに動く。触れただけ・はじいただけではコントロールではない。

- チームコントロールの開始：プレーヤーがライブのボールを**持つ／ドリブルする／与えられた**とき
- 継続：コントロール中・味方同士のパス中
- 終了：相手がコントロールした／ボールがデッドになった／ショットでボールが手から離れたとき
- **ディフェンスが片手でタップしただけではコントロールは変わらない**（両手で保持・片手で保持したら変わる）
- スローインは、スローインするプレーヤーが持った時点でコントロール済み → コート上のプレーヤーが**触れた瞬間**に計測開始

## 操作の基本ルール（一般）

### スタート／再開
- コート上でプレーヤーがライブのボールをコントロールしたとき
- スローインのボールにコート上のプレーヤーが触れたとき

### 止めるが**リセットしない**（残り秒数を表示したまま）
それまでボールをコントロールしていたチームに引き続きスローインが与えられるとき：
- ボールがアウトオブバウンズになった（**相手が最後に触れて出た**場合を含む）
- コントロールチームのプレーヤーの怪我でゲームが止まった
- **コントロールチームに**テクニカルファウルが宣せられた
- ジャンプボールシチュエーション／ダブルファウル／罰則の相殺
- （一般のみ）フロントコートのスローインで残りが**14秒以上**あるとき

### 止めて**24秒にリセット**（表示は非表示 ※消せない機材は24を表示）
- ボールが正当にバスケットに入ったとき
- ボールがリングに触れて**相手チーム**がコントロールしたとき
- 新たなチームがコントロールを得たとき（スティール等）→ 即リセット＋スタート
- ファウル・バイオレーションで**相手チーム**にスローインが与えられるとき（一般はバックコートからのとき）
- フリースローを行うとき（例外：攻撃側のテクニカルファウルの罰則FTは**継続**のまま行う）

### **14秒にリセット**（14を表示）
- ショット（パスが偶然リングに当たった場合を含む）や最後のFTが**リングに触れて不成功**となり、**攻撃していたチームが再びコントロール**したとき（リングとバックボードに挟まった場合を含む）
- ボールがリングに挟まったジャンプボールシチュエーションで攻撃側にボールが与えられるとき
- （一般のみ）同じチームが引き続き**フロントコート**からスローインし、残りが**13秒以下**のとき
- （一般のみ）**それまでコントロールしていなかったチーム**が、パーソナルファウル／バイオレーション／JBSの結果**フロントコート**からスローインを与えられるとき
- （一般のみ）アンスポ・DQの罰則でスローインラインからスローインするとき

### ⚠️ U12の違い（フロントコート・スローインラインがないため）

| 場面 | 一般 | U12 |
|------|------|-----|
| 相手のファウル／バイオレーション後のスローイン | フロントコートなら14秒（13秒以下時）または継続（14秒以上時） | **常に24秒リセット** |
| アンスポ・DQの罰則後 | スローインラインから・14秒 | **センターライン延長から・24秒** |
| 14秒リセットになるケース | 上記含め複数 | **「リングに触れて攻撃側が再コントロール」と「リングに挟まったJBSで攻撃側」の2つだけ** |

## 実務の鉄則

- **審判の笛 → まずストップ → 審判のレポート／シグナルを確認 → 継続かリセットかを判断**（いきなりリセットしない）
- リセットする前に**残り秒数を覚えておく**（メモ可）。誤ったときに戻せる根拠になる
- 審判から**リセットのシグナル**（人差し指を回す）があれば直ちに従う。「リセットか継続か」でタイマーと見解が割れたら、審判のジェスチャーを確認し、なければショットクロックオペレーターが判断する
- ボールがリングに触れた／入った瞬間に「当たった」「入った」とコールして非表示にする
- スタート／ストップは**タイマーと同時にコール**して目視確認。同時にならないのは2場面のみ：①ジャンプボール後（コントロール成立を待つ）②最後のFT不成功でリングに触れたあと（コントロール確立で14/24を判断）
- スローインの前に残り秒数をクルーに知らせる（例：「継続8秒」）

## Q・OT終了間際の表示

- ゲームクロック残りが**24秒未満〜14秒**：リセット場面が生じたら24にリセットして**非表示**を続ける。その後攻撃側リバウンドがあれば14にリセットしてスタート
- ゲームクロック残り**14秒未満**：リセット場面が生じたら競技時間終了まで**非表示**のまま
- 非表示にできない機材は**24を表示**しておく。各Q・OTの開始時は24を表示

## ショットクロックの誤り・特殊ケース

- **誤ってリセットしてしまった**：表示を止めて（非表示または24）、その瞬間の**ゲームクロックの残り時間を記憶・メモ**し、ストップウォッチを動かす。審判が気づかなければ、ボールがデッドになってから知らせる（プレー中にブザーで止めてはいけない）
- **ブザーが誤って鳴った**：ブザーは無視されゲーム続行。コントロールしているチームが不利になった場合のみ、審判がゲームを止めて訂正する
- ショットのボールが空中にある間にブザーが鳴った：入れば得点、リングに当たれば続行、当たらず審判が笛 → バイオレーション（相手が速やかに明らかにコントロールした場合、審判は笛を鳴らさない）
- デジタル表示がない場合（U12の小旗運用）：残り10〜5秒は**黄色**、残り5〜0秒は**赤色**の小旗で表示
