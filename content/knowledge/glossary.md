# 用語集

last_updated: 2026-07-13

| 用語（日） | 英語 | 意味 |
|----------|------|------|
| バイオレーション | Violation | ルール違反（ファウル以外）。相手のスローインで再開。 |
| ファウル | Foul | 不正な身体接触または反スポーツ的行為 |
| パーソナルファウル | Personal Foul | プレーヤー間の身体接触によるファウル |
| テクニカルファウル | Technical Foul | 接触なし。態度・言動・規則外行為 |
| アンスポーツマンライクファウル | Unsportsmanlike Foul (U) | 過度な・不必要な接触ファウル（⚠️ FIBA 2026改定で下記DI/FLに分割予定。国内未適用 → [[07-to-basics]]の予告参照） |
| ディスラプティブファウル | Disruptive Foul (DI) | ⚠️ FIBA 2026新設（国内未適用）。ゲームの流れを止める不必要な接触。失格（GD）の対象外 |
| フレグラントファウル | Flagrant Foul (FL) | ⚠️ FIBA 2026新設（国内未適用）。乱暴・危険な高強度の接触。失格（GD）の対象 |
| ディスクォリファイングファウル | Disqualifying Foul | 著しく反スポーツ的行為。失格退場 |
| チャージング | Charging | 攻撃側のボールを持つプレーヤーによる違法な接触 |
| ブロッキング | Blocking | ディフェンスによる違法な接触 |
| ピボット | Pivot | 一方の足を軸として他方の足を動かす動作 |
| ギャザーステップ | Gather Step | ドリブル終了時の最初の一歩（ステップカウントに含まない） |
| シリンダー | Cylinder | プレーヤーの占有空間（円筒形） |
| マンツーマン | Man-to-Man | 各選手が特定の相手をマークするディフェンス |
| チームファウル | Team Foul | 1ピリオドに積み上がるチームのファウル累計 |
| ボーナス | Bonus | チームファウルが4個以降、相手がフリースローを得られる状態 |
| オルタネイティングポゼッション | Alternating Possession | ジャンプボール後の交互ポゼッション制度 |
| デッドボール | Dead Ball | プレーが止まっている状態 |
| ライブボール | Live Ball | プレーが行われている状態 |
| スローイン | Throw-in | ラインの外からのパスで再開 |
| リードオフィシャル | Lead Official | エンドライン側の審判 |
| トレイルオフィシャル | Trail Official | ハーフライン側の審判 |
| テーブルオフィシャルズ (TO) | Table Officials | スコアラー・タイマー等、卓上でゲームを記録・管理する4人（→ [[07-to-basics]]） |
| スコアラー | Scorer | スコアシートを記録するTOの主担当。タイムアウト・交代の合図とAPアロー操作も担う |
| アシスタントスコアラー | Assistant Scorer | スコアボードを操作し、ファウル個数を表示するTO |
| タイマー | Timer | ゲームクロック（競技時間・インターバル・タイムアウト）を計るTO |
| ショットクロックオペレーター | Shot Clock Operator | ショットクロック（24秒ルール）を操作するTO |
| TO主任 | TO Chief | TOクルーを支援するオフィシャルチームの一員（競技規則上のコミッショナーとは別） |
| ゲームクロック | Game Clock | 競技時間を表示するクロック |
| ランニングスコア | Running Score | スコアシート上の得点経過の記録欄 |
| APアロー | Alternating Possession Arrow | 次のAPスローインの権利を示す矢印の表示器具 |
| GD | Game Disqualification | ファウルの累積による失格・退場を表すスコアシート記号 |
| L2M | Last 2 Minutes | 第4Q・OTの最後の2分間（一般のみ特別規定あり。U12は適用外） |
| クルーチーフ | Crew Chief | 審判団の責任者（主審） |
| コミッショナー | Commissioner | TOの仕事を監督し審判をサポートする役員（同席する場合） |
| ショットクロック | Shot Clock | 攻撃制限時間（一般は24秒） |
| 制限区域 | Restricted Area / Paint | ゴール下の長方形ゾーン（3秒ルール適用） |
