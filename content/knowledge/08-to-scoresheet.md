## スコアシートの構成

スコアシートは**ゲームの公式な記録**。3つの部分からなる：

1. **ヘッダー部分** — 大会名・日付・時間・会場・ゲームナンバー・審判氏名（ゲーム後にTOクルーと審判がサイン）
2. **チーム欄** — メンバーリスト、出場記録、ファウル、チームファウル、タイムアウト
3. **ランニングスコアと最終結果** — 得点経過、各Q得点、最終スコア、勝者チーム、終了時刻

**用具**：濃い色（青/黒）と赤のペン、定規、笛（ブザー故障時用）、予備スコアシート、メモ用紙

### ペンの色ルール（最重要の基本）

| 色 | 使う場面 |
|----|---------|
| **赤** | 第1Q・第3Qのランニングスコア／ファウル／タイムアウト／チームファウル、スターターの×を囲む○ |
| **濃色（青/黒）** | 第2Q・第4Q・**OT**の同項目、ヘッダー・メンバー等の事前記入、締めの線 |

> OTは「第4Qに起こったもの」とみなすため濃色。プレーのインターバル中のファウルは**次のQの色**で記録する。

## ゲーム開始前の記入

- チームリスト（開始40分前、U12国内は30分前まで提出）をもとに、**ユニフォーム番号順**に氏名を記入
- 先に記載されたチーム（ホーム）がチームA＝上の枠、チームBが下の枠
- キャプテンは氏名の横に**（CAP）**（U12はキャプテン規定なし）
- 使わなかった行は横線＋斜線で消す（HCのサイン後に引く）
- 開始10分前まで：HCがスターター5人の番号横（Player in欄）に**×**を記入してサイン（チームAが先）
- ゲーム開始時：スコアラーがコート上の5人を確認し、**×を赤ペンの○で囲む**。相違があれば速やかに審判へ
- 交代要員が初出場したら Player in 欄に×（○では囲まない。そのQの色で）

> U12は**出場時限欄**を使う：各Qの開始前にメンバーチェックし、出場するプレーヤーに **／** を記入。途中出場は **＼**。HCのサインは不要（出場義務ルールのチェックに直結するので正確に）。

## 得点の記録（ランニングスコア）

| 得点 | 記入方法 |
|------|---------|
| フリースロー（1点） | 数字に**塗りつぶした丸●**＋隣にプレーヤー番号 |
| フィールドゴール（2点） | 数字に**斜線／**（左利きは＼でも可）＋隣に番号 |
| 3点（一般） | 斜線＋**番号を○で囲む** |
| オウンゴール（一般） | 相手チームの**そのときコート上のキャプテン**の得点として記入 |
| オウンゴール（**U12**） | 番号の代わりに **▲** を記入 |

- ゴールテンディング／インタフェアレンスで認められた得点は、ショットしたプレーヤーの得点として記入
- U12は3点なし（採用する大会もある。その場合は一般と同じく番号を○で囲む）
- **スリーポイントかどうかはスコアラーが判断しない**。審判のシグナルで判断する

### Q終了・ゲーム終了の締め

- 各Qの終わり：そのQのペンで最後の得点を**太い○**で囲み、得点と番号のすぐ下に**太い横線1本**。下段「スコア」欄にそのQの得点を記入
- 未使用のファウル枠に2本の横線、前半終了時は使用済み／未使用ファウル枠の間に**濃色の太線**
- ゲーム終了：最後の得点を○で囲み**太い横線2本**、未使用のランニングスコア列に斜線（濃色）、最終スコア・勝者チーム名・**終了時刻**を記入
- 毎Q終了時に、ランニングスコア・ファウル数・タイムアウト数を**スコアボードと照合**する

## ファウルの記録

### 記号一覧

| 記号 | 意味 |
|------|------|
| **P** | パーソナルファウル |
| **T** | テクニカルファウル（プレーヤー） |
| **U** | アンスポーツマンライクファウル |
| **D** | ディスクォリファイングファウル |
| **F** | 第39条ファイティングによるDQファウル |
| **C** | HC自身の言動によるテクニカルファウル |
| **B** | ベンチ（A.コーチ・交代要員・チーム関係者等）の言動によりHCに宣せられるテクニカルファウル |
| **M** | マンツーマンペナルティ（**U12・U15のみ**。HCに記録） |
| **GD** | 失格・退場（Game Disqualification）。該当ファウルのすぐ隣の枠（5個目なら欄外）に記入 |

- フリースローが与えられるときは記号の右下に本数を小さく添える（例：P₂、U₂、T₁、B₁）
- 罰則が相殺されたファウル（第42条）は記号の後ろに小さい **c**
- **HCに記録されるファウル（C・B・M）はチームファウルに数えない**
- スローインファウル（第4Q・OT最後の2分のスローイン中のディフェンスファウル）は **P₁** と記録し、**チームファウルに数える**

### 失格・退場（GD）になる組み合わせ

| 対象 | 組み合わせ |
|------|-----------|
| プレーヤー | 2個のT ／ 2個のU ／ T＋U |
| HC | **2個のC** ／ **3個のB** ／ **2B＋1C** ／ **3個のM**（U12。U15はM2個） |

- 記録したら**速やかに審判に知らせる**（5個目のファウルも同様）
- 交代要員・5ファウル済みメンバー・チーム関係者のT／DQは、HCの枠に**B**として記録（DQ本人にはD。このBはチームファウルに数えない）

### チームファウル欄

- プレーヤーのファウル（P/T/U/D）が記録されるたびに、1〜4の枠を順に**大きい×**で消す
- 各Qの終わりに未使用の枠へ2本線。**OTのファウルは第4Qに続けて数える**
- 4個目のチームファウルが宣せられたら、ボールがライブになってから**チームファウルペナルティの赤色標識**を表示（国内はアシスタントスコアラー担当）
- 記録のたびに「白14番、個人4個目、チーム3個目」のように**声に出して**クルーと共有する（伝えるチームファウル数は5個まで）

## タイムアウトの記録と合図

- 記録：認められたときの**各Q経過時間（分・切り上げ）**を枠に記入
  - 例（U12・6分Q）：残り2分35秒 → 「4」（6−2.35→切り上げ）
  - 例（一般・10分Q）：残り3分44秒 → 「7」
- 未使用の枠は各ハーフ（U12は各Q）の終わりに2本線で消す
- 回数：一般は前半2回・後半3回（第4Q残り2:00以下では2回まで）・OT各1回。**U12は各Q・各OTに1回ずつ**

### 請求から成立までの流れ

1. HCまたはA.コーチが請求（**条件付き請求「入ったらタイムアウト」は認められない**）
2. スコアラーは「**タイムアウトが認められる時機**」にブザー＋Tのジェスチャー＋請求チームのベンチを指す
3. 審判が笛＋シグナルを示したらタイムアウト開始（タイマーが計測）
4. 審判が確認したらスコアシートに記入

「認められる時機」の要点：
- ボールがデッドでゲームクロックが止まったとき（ファウル時は審判のレポート後）→ **両チーム可**
- 相手のフィールドゴール成功後 → **得点されたチームのみ可**（請求の先後より優先）
- スローイン／最初のFTのボールがプレーヤーに与えられたら**終了**。それ以降のブザーは遅すぎ（認められない）
- 残っていないチームが請求したら、審判を通じてコーチに伝える

### 交代の合図（国内はスコアラーの担当）

- **交代要員自身**がテーブルに申し出る（HC・A.コーチの申し出は不可）
- ブザー＋両手を顔の前で交差する合図＋チームのベンチを指す
- ブザーを鳴らす前なら申し出の取り消し可
- 交代・タイムアウトの時機の詳細（U12はQ1〜Q3インターバルのみ等）→ [[04-minibasket-diff]]

## オルタネイティングポゼッション（AP）アロー

スコアラーの担当。**次のAPスローインの権利を持つチームのバスケット（攻める方向）に矢印を向ける**。

| 場面 | 操作 |
|------|------|
| ゲーム開始のジャンプボール後 | コントロールを得られなかったチームの攻撃方向にアローをセット |
| APスローインが終わったとき（ボールがコート上のプレーヤーに正当に触れた／スローインチームのバイオレーション） | **速やかに反転** |
| 前半が終了したとき | 攻撃方向が変わるため**速やかに反転**し、審判（・両チーム）と共有 |
| APスローインの前・最中に**ファウル**があったとき | **反転しない**（権利は取り消されない） |

- ジャンプボールシチュエーションが宣せられたら（第2Q以降・OTの最初のスローイン前を含む）、**アローに手を添えて反転の準備**をする。高く持ち上げて審判に見せる必要はない
- 誤った向きでスローインしてしまった場合、**ボールがコート上のプレーヤーに正当に触れた後は訂正できない**（ただし権利は失われない）→ [[10-to-difficult-cases]]

## 誤りの訂正

- **訂正できる期限：ゲーム終了の合図の後でも、クルーチーフがスコアシートにサインする前まで**（結果に影響しても訂正しなければならない）。サイン後は訂正不可（大会主催者への報告のみ）
- ゲーム中に誤りを見つけたら：次にボールがデッドでゲームクロックが止まったときにブザーで審判に知らせる
- 国内大会の訂正方法：**間違いを二本の横線で消し、周囲の余白に正しい記録**（修正テープ・修正液は不可）。クルーチーフの小さなサインは省略できる
- 代表的なケース：
  - **得点を少なく記録**（2点と書いたが3点だった）→ 横線で消して正しく記録
  - **得点を多く／誤ったチームに記録** → 消さずに正しい得点から記録を再開し、誤った得点の横に**小さい点（・）**を付けて目印にする
  - **番号・ファウル種類の間違い** → 正しい記号を間違いの上に重ねて記入し、横に小さい点（・）
- すぐ直せないときは**付箋**を貼り、タイムアウトやインターバルで訂正する

## アシスタントスコアラーの仕事

- **スコアボードの操作**。スコアボードとスコアシートに相違があり解決できないときは**スコアシートが優先**
- ファウル個数の表示（国内の分担）：スコアラーのコール後、標識を**①ベンチ→②正面→③再びベンチ**の3段階で表示（5個目はブザー〔スコアラー〕＋2段階表示）
- チームファウルの表示：審判のレポートで確認したら**個人ファウルより先に**表示。積み上げ式器具の場合、チームA側はショットクロックオペレーターが担当
- 4個目のチームファウル後：ボールがライブになったら赤色標識を表示（数字は消す）
- **5個目のファウルのプレーヤーを審判が見逃してスローインさせようとしているときは、ブザーで知らせる**
- 得点入力とゲームクロックが一体の機材では**タイマーが得点入力**を行う（その場合もクロック操作優先。得点コールはAスコアラーが行う）

### スコアラー系の声出し（U12ハンドブックの型）

| 場面 | スコアラー | Aスコアラー |
|------|-----------|------------|
| ショット | 「赤4番。カウント／アウト」「赤4番カウント。23点目」 | 「赤4番。19対23」（**常にA対Bの順**） |
| ファウル | 審判と同時復唱＋サムアップ →「白4番。個人3個目。チーム4個目」 | 表示して復唱＋OK |
| タイムアウト | ブザー＋合図 →「白のタイムアウト」 | 「白のタイムアウト、OK」 |
| 交代 | ブザー＋合図 →「白33番イン」 | 「白33番イン、OK」 |

ショット成功時の手順の習慣：**ベンチ確認（タイムアウト請求）→ ライブ確認 → 記入**

## 一般とU12のスコアシートの違い（まとめ）

| 項目 | 一般 | U12 |
|------|------|-----|
| 様式 | 4枚1組複写式 | **1枚もの・専用様式** |
| キャプテン | (CAP)を記入 | 規定なし |
| 出場記録 | Player in欄に×（スターターは○囲み） | **出場時限欄**に／（途中出場は＼）。コーチのサイン不要 |
| HCのファウル記号 | C・B | C・B＋**M**（M3個でGD） |
| 3点 | あり | なし（採用大会のみあり） |
| オウンゴール | 相手キャプテンの得点 | **▲** |
| タイムアウト | 前半2・後半3・OT1 | **各Q・各OTに1回** |
