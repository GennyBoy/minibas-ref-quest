## TOとは

テーブルオフィシャルズ（TO）は、審判・コミッショナーとともに**オフィシャルチーム**を構成する一員。ゲーム中に起こったことを記録し、ゲーム管理に必要な電子機器を操作する。

**4人で構成**（各1人）：

| 役割 | 主な仕事 |
|------|---------|
| スコアラー | スコアシートの記録、審判への合図（タイムアウト・交代）、APアローの表示 |
| アシスタントスコアラー | スコアボードの操作、ファウルの表示、スコアラーのサポート |
| タイマー | ゲームクロックの操作（競技時間・インターバル・タイムアウトの計測） |
| ショットクロックオペレーター | ショットクロックの操作（24秒ルールの適用） |

- U12・U15のゲームでは、大会主催者の判断により**補助員1名を追加**できる（5人体制）
- 座席はコート片方のチームベンチの間、テーブル中央部。U12ハンドブックの配置は Aチームベンチ側から **ショットクロックオペレーター → タイマー → スコアラー → Aスコアラー**
- U12では後ろに**TO主任**または**TOサポーター**の席を設けることがある（後述）

### よくある誤解：「アシスタントスコアラーはもういない」？

**誤り。現行の競技規則に明記されている**（2026-07-13 一次資料で確認）：

- JBA競技規則には**第48条「スコアラー、アシスタントスコアラー：任務」**が現存する。第49条の【補足】には「国内大会においてはアシスタントスコアラーが行う」という国内向けの分担（プレーヤーファウルの表示器具の操作、GDマーカーの提示、チームファウル表示など）が複数箇所に明記されている（[2025年変更点パート2](https://www.japanbasketball.jp/files/referee/rule/2025rule_changepoint2_20250128.pdf)）
- [JBA 2026年変更点](https://www.japanbasketball.jp/files/referee/rule/2026fibarule_changepoint_20260401.pdf)は「2025年度の競技規則改訂から**変更なし**」で、TO構成の変更はない
- [FIBA OBR 2026 改定（2026年10月1日施行）](https://assets.fiba.basketball/image/upload/documents-corporate-fiba-official-rules-2026-rule-changes-v10a.pdf)にもTO構成・第48条の変更は**含まれない**
- 誤解の出どころはおそらく**ローカル大会の運用**：ミニバスの県大会・交流戦などではTOを3人（スコアラー・タイマー・SCオペレーター）や2人で回し、Aスコアラーの仕事を他が兼務することが多い。「その大会にAスコアラー席がない」≠「規則から消えた」

## ゲーム開始前の任務（第3章）

### 到着〜準備
- FIBA大会はゲーム開始90分前までに会場入り（国内は大会主催者の指示による）
- 到着したらまず大会主催者・コミッショナーに知らせ、オフィシャルチームと合流
- **TOミーティング**を行う：開始時刻、規則の最新変更点、重大な場面（得点・タイムアウト・交代・Q終了・チームファウル・APの手順）での対応方法、機器の位置と誤作動時の対応、審判とのコミュニケーション方法、を確認する

### 時系列チェックリスト（FIBA基準。国内・U12は主催者判断で変更あり）

| タイミング | やること |
|-----------|---------|
| 開始40分前まで | チームリスト提出を受ける（**U12国内は30分前**） |
| 審判コート入り前 | TOはテーブルにつく。機器（ゲームクロック・ショットクロック・ブザー・スコアボード）点検 |
| 審判がコートにいる間 | ショットクロックを最後まで進め、**ブザー音を審判に聞かせる** |
| 開始10分前まで | 両HCがメンバー・スターター5人を確認し**スコアシートにサイン**（チームA → B の順） |
| 開始3分前 | タイマーがブザー（審判は3本指で3分前を示す） |
| 開始1分30秒前 | タイマーがブザー（審判が笛。両チームはベンチエリアに揃う） |

## ゲーム中の共通原則

- **Q終了間際とゲーム最後の2分間は特に集中力を上げる**
- ゲームの流れを注視し、交代・タイムアウトの請求を**予測**する（特に得点後のタイムアウト請求）
- **ブザーは慎重に使う**。ブザーはゲームクロックやゲームを止めるものではなく、ボールをデッドにするものでもない
- **審判の判定に異を唱えるシグナルは絶対に送らない**。不明な点は審判に短く明確に告げて確認する
- 審判を「判断のできない状況」に追い込まない：審判をテーブルに呼ぶ前に、**何が起きたか正確に把握してから**伝える
- 審判から不明確な状況（Q終わりの得点等）の確認要請があったら、**事実のみを1人が代表して**伝える。ジェスチャーや大声は使わない
- 機器の誤作動は、規則に定められたタイミング（ボールデッド時）で審判に知らせる

### ファイティング発生時の視野分担
コート上でファイティングが起きた／ベンチパーソネルがベンチを離れたとき：

- アシスタントスコアラー → **アウェーチームのベンチ**を注視
- ショットクロックオペレーター → **ホームチームのベンチ**を注視
- スコアラーとタイマー → **コート上**を注視
- 該当者と時間を個々に記録する（審判の報告書作成をサポートするため）

## 審判とのコミュニケーション

- ファウルのレポートは**アイコンタクト → 同時復唱 → サムアップ（OKサイン）**で受ける
- 覚えるべき審判のシグナル：
  - **手を振り下ろす** → ゲームクロック／ショットクロックをスタート
  - **サムアップ（親指立てる）** → OK・コミュニケーション
  - **人差し指を伸ばして手を回す** → ショットクロックのリセット（コントロールが変わった／ボールがリングに触れた）
- タイムアウト・交代の合図は**座ったまま、ブザーとジェスチャーだけで**行う（声は出さない）。声出しはTOクルー同士のコミュニケーション用

## ゲーム終了後の任務

1. スコアラーがスコアシートを完成させる（終了時刻の記入を含む）
2. **TO全員がサイン** → アンパイアがサイン → **クルーチーフが最後に点検・承認してサイン**（この順番）
3. クルーチーフのサインで、そのゲームに対するオフィシャルチームの管理権限が終了する
4. ゲーム終了後に何か起きたら観察・記録し、審判の報告書作成を手伝う
5. スコアシートのコピーをチームに返却
6. 会場を離れる前に審判・コミッショナーに許可を求める

> ⚠️ FIBAの「抗議の手続き」（終了後15分間テーブルを離れない等）は**国内大会では原則採用しない**（JBA承認がある場合のみ）。

## U12でのTO運用（TO主任・TOサポーター）

U12ではプレーヤー（子どもたち）自身がTOを務めることが推奨されており、それを支える大人の役割が定義されている：

| 役割 | 位置づけ |
|------|---------|
| **TO主任** | TOクルーが役割を果たせるよう支援するオフィシャルチームの一員。準備・ミーティング・ゲーム中のサポート・トラブル対応（競技規則第45条のコミッショナーとは別の立場） |
| **TOサポーター** | U12でTOを務める子どもたちをテーブルの後ろから見守り、手助けする大人 |

共通の心得：
- TOは**ニュートラル**。応援や感情表現はしない
- スコアラーの代わりにジェスチャーやブザーを行うことは**厳に慎む**（例外：L2Mでゴール後のタイムアウト／交代請求にスコアラーが対応できないときのみ、TO主任がブザーを鳴らせる）
- 対処に困るトラブルは、プレーが止まっているときに審判を呼ぶ。**プレー中に審判を呼ぶブザーは鳴らさない**
- ミスは責めずに原因分析と再発防止。前向きな声かけをする

具体的なトラブル対応は [[10-to-difficult-cases]] を参照。

## ⚠️ 予告：FIBA 2026年改定（2026年10月1日施行、国内適用は未発表）

[FIBA OBR 2026 Rule Changes](https://assets.fiba.basketball/image/upload/documents-corporate-fiba-official-rules-2026-rule-changes-v10a.pdf)（2026年6月発行）で、ファウル体系とスコアシート記号が大きく変わる：**アンスポーツマンライクファウルが廃止**され、ディスラプティブファウル（DI・失格対象外）とフレグラントファウル（FL・失格対象）に分割。テクニカルファウルもカテゴリー1/2に分類され、記号は DI／⭕FL／⭕T・T／⭕BD 等に変更。TO構成の変更はなし。国内適用は例年どおりなら2027年度から → 発表されたら `/rule-diff 2027` で本対応する。
